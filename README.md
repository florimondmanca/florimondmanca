## Install

If you intend to use the Angular CLI, install it globally now:

```bash
$ npm install -g @angular/cli
```

Then, install the dependencies:

```bash
$ npm install
```

You can now start the development server, which will run on `http://localhost:4200/`:

```bash
$ ng serve
```
